## Опис проєкту

**DistLearn.WPF** — це desktop-застосунок платформи дистанційного навчання, розроблений у межах курсового проєкту з дисципліни **«Об’єктно-орієнтоване програмування»**.

Програма реалізує базовий функціонал системи дистанційного навчання: перегляд курсів, реєстрацію та авторизацію користувачів, роботу з навчальними матеріалами і завданнями, здачу робіт студентом, перегляд файлів викладачем, а також керування користувачами і курсами адміністратором.

## Мета проєкту

Метою проєкту є розроблення програмного забезпечення для предметної області **платформа дистанційного навчання** з використанням об’єктно-орієнтованої парадигми, графічного інтерфейсу користувача та збереження даних між сеансами роботи програми.

## Основний функціонал

У програмі реалізовано:

- перегляд списку курсів;
- перегляд загальної інформації про курс;
- реєстрацію нового користувача;
- авторизацію користувача;
- окремі кабінети для студента, викладача та адміністратора;
- відкриття сторінки курсу;
- перегляд навчальних матеріалів;
- перегляд завдань;
- здачу роботи студентом у вигляді файлу або коментаря;
- перегляд зданих робіт викладачем;
- створення курсів викладачем;
- додавання і видалення матеріалів;
- додавання і видалення завдань;
- додавання і видалення студентів із курсу;
- додавання і видалення користувачів адміністратором;
- видалення курсів адміністратором;
- збереження і завантаження даних у форматі JSON;
- автоматичне збереження даних після закриття програми;
- автоматичне завантаження даних після повторного запуску.

## Ролі користувачів

### Гість
Гість може:
- переглядати список курсів;
- переглядати загальну інформацію про курс;
- відкривати форму реєстрації;
- створювати новий обліковий запис.

### Студент
Студент може:
- авторизуватися в системі;
- переглядати свої курси;
- переглядати матеріали курсу;
- переглядати завдання;
- відкривати сторінку завдання;
- додавати файл або коментар до відповіді;
- здавати роботу на перевірку.

### Викладач
Викладач може:
- авторизуватися в системі;
- переглядати власні курси;
- створювати нові курси;
- додавати матеріали;
- додавати завдання;
- встановлювати дедлайн для завдань;
- відкривати матеріали та завдання;
- додавати студентів на курс;
- видаляти студентів із курсу;
- видаляти матеріали;
- видаляти завдання;
- видаляти курс;
- переглядати здані студентами роботи;
- відкривати прикріплені файли.

### Адміністратор
Адміністратор може:
- авторизуватися в системі;
- переглядати список користувачів;
- додавати нових користувачів;
- видаляти користувачів;
- переглядати список курсів;
- видаляти курси;
- зберігати дані у JSON;
- завантажувати дані з JSON.

## Основні вікна програми

У проєкті реалізовані такі вікна:

- **MainWindow** — головне вікно зі списком курсів;
- **LoginWindow** — вікно авторизації;
- **RegisterWindow** — вікно реєстрації;
- **StudentWindow** — кабінет студента;
- **TeacherWindow** — кабінет викладача;
- **AdminWindow** — панель адміністратора;
- **CourseWindow** — сторінка курсу;
- **MaterialWindow** — вікно перегляду матеріалу;
- **AssignmentWindow** — вікно перегляду завдання та здачі роботи.

Для запуску проєкту потрібно:

1. Відкрити рішення у **Visual Studio 2022**.
2. Обрати проєкт **DistLearn.WPF** як стартовий.
3. Запустити програму.
